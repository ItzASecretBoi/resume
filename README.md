  Simple FPS Template was originally intended to be, well... a simple FPS template, but during development I ended up becoming absolutely obsessed with inventory systems, so I made one.
  I wanted to build a hotbar inventory system similar to Minecraft, and while it can definitely be improved, it taught me a lot about data organization and how to structure inventory systems for games.
  I also created a scrolling interaction system where players can cycle between multiple interactions on objects like items or doors and trigger different actions based on what they select.
